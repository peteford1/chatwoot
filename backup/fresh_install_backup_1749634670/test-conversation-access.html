🗨️ Conversation Center Access Guide

📋 Current System Status

  • ✅ Laravel Server: Running on port 8000
  • ✅ WebSocket Server: Running on port 8001
  • ✅ Database: Connected with Docker MySQL
  • ✅ Store User: Created and ready

🔑 Step 1: Login to App Panel

URL: http://localhost:8000/login

Credentials:

  • Email: store@example.com
  • Password: password

🏠 Step 2: Navigate to Dashboard

After login, you should see the main dashboard with a sidebar navigation.

💬 Step 3: Access Conversation Center

Look for "Conversations" in the left sidebar and click it.

Direct URL: http://localhost:8000/conversation-center

🎯 What You Should See

  • 📋 Left sidebar with conversation list (5 mock conversations)
  • 💬 Right side chat interface
  • 🔗 WebSocket connection status indicator
  • ➕ "New" button to start conversations
  • 🔍 Search bar for filtering conversations

🔧 Alternative Access Methods

Admin Panel Access:

  • URL: http://localhost:8000/cp
  • Email: admin@voicelinkai.com or admin@example.com
  • Password: password
  • Note: Admin panel doesn't have conversation center

🐛 Troubleshooting

  • If you see "Not Found": Make sure you're logged in as store@example.com
  • If login fails: Check that the password is exactly password
  • If no conversations load: Check browser console for JavaScript errors
  • If WebSocket fails: Verify port 8001 is not blocked
